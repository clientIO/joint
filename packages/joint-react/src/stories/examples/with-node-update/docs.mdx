# Updating Nodes Dynamically

This guide demonstrates various ways to update nodes (elements) in your diagram using the `useGraph` hook. Learn how to modify element properties, add new elements, and remove existing ones.

## Overview

The `useGraph` hook provides methods to modify the graph state:
- **setElement**: Update existing element properties
- **removeElement**: Remove elements from the graph
- **setLink**: Update existing link properties
- **removeLink**: Remove links from the graph

## Key Patterns

### Updating Properties

```tsx
const { setElement } = useGraph();

// Update a single property
setElement('element-id', (prev) => ({ ...prev, label: 'New Label' }));

// Update multiple properties
setElement('element-id', (prev) => ({
  ...prev,
  x: 100,
  y: 200,
  color: 'blue'
}));
```

### Adding Elements

```tsx
const { setElement } = useGraph();

setElement('new-element', {
  x: 100,
  y: 100,
  width: 100,
  height: 50,
});
```

### Removing Elements

```tsx
const { removeElement } = useGraph();

removeElement('element-id');
```

## Best Practices

1. **Immutable updates**: Always return new objects in updater functions
2. **Type safety**: Use TypeScript to ensure correct property types
3. **Batch updates**: Consider batching multiple updates for performance
4. **Error handling**: Validate element IDs before updating
