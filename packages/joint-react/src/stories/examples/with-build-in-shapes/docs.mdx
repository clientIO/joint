# Render build in [standard `jointjs` shapes](https://docs.jointjs.com/learn/features/shapes/built-in-shapes/standard)
When using JointJS shapes namespace contains a set of standard shapes. 
You can find specifications on every shape in our [API reference](https://docs.jointjs.com/api/shapes/standard/). 

We do not need to use RenderElement, as we are not using **react components**.
Here are examples:
