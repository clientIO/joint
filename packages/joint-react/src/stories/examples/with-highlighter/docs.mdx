# Using Highlighters

This example demonstrates how to use highlighters to create visual emphasis on elements in your diagram. Highlighters are React components that wrap your element content and add visual effects like borders, opacity changes, or custom styling.

## Overview

Highlighters are useful for:
- **Hover effects**: Show visual feedback when users interact with elements
- **Selection states**: Indicate which elements are currently selected
- **Focus indicators**: Highlight elements that have focus or are being edited
- **Status indicators**: Show different states (active, disabled, error, etc.)

## Available Highlighter Types

### Mask Highlighter
Creates a border/mask around the element. Perfect for selection indicators.

```tsx
import { Highlighter } from '@joint/react';

<Highlighter.Mask 
  stroke="blue" 
  strokeWidth={2} 
  padding={5}
>
  <YourElement />
</Highlighter.Mask>
```

### Opacity Highlighter
Changes the opacity of the element. Great for hover effects.

```tsx
import { Highlighter } from '@joint/react';

<Highlighter.Opacity opacity={0.5}>
  <YourElement />
</Highlighter.Opacity>
```

### Stroke Highlighter
Adds a stroke outline around the element.

```tsx
import { Highlighter } from '@joint/react';

<Highlighter.Stroke 
  stroke="red" 
  strokeWidth={3}
>
  <YourElement />
</Highlighter.Stroke>
```

### Custom Highlighter
Create your own custom highlighting effect.

```tsx
import { Highlighter } from '@joint/react';
import { highlighters } from '@joint/core';

<Highlighter.Custom 
  onAdd={(elementView) => {
    return highlighters.mask.add(elementView, {
      padding: 10,
      rx: 5,
      ry: 5,
    });
  }}
>
  <YourElement />
</Highlighter.Custom>
```

## Best Practices

1. **Use forwardRef**: Child elements must forward refs for highlighters to work
2. **Conditional rendering**: Use `isHidden` prop to show/hide highlighters dynamically
3. **Performance**: Highlighters are lightweight and don't impact rendering performance
4. **Accessibility**: Consider color contrast and visual indicators for accessibility
