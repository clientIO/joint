# Render Link Example

This example demonstrates how to use `renderLink` to render custom React components for links. Each link is rendered as an SVG `<path>` with a label placed at the midpoint using a `<foreignObject>`.

The `useLinkLayout` hook provides layout data (source/target coordinates and path `d` attribute) for positioning the link path and label.
