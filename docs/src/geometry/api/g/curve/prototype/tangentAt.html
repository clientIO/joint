curve.tangentAt(ratio [, opt])

Return a line that is tangent to the curve at a point that lies ratio (normalized length) away from the beginning of the curve.

The function expects ratio to lie between 0 and 1; values outside the range are constrained to 0 and 1, respectively.

The tangent line starts at the specified point. The direction from start to end is the same as the direction of the curve at the specified point. If the control points of the curve all lie at the same coordinates, null is returned (it is impossible to determine the slope of a point).
