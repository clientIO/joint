The 'anchor' connection point function is the simplest connection point function. It places the connection point at the link end's anchor point (determined either by the anchor function or by the defaultAnchor paper option). It accepts a few additional arguments, which can be passed within the connectionPoint.args property:

offset number | object An object with x and y properties. The connection point will be moved:
  • from the anchor point by x points in the direction of the first link path end segment.
  • by y points in the direction of the first link path end segment rotated by 90 degrees anti-clockwise around the anchor point.
If a number is provided, it will be used as the x offset.
align 'top' | 'bottom' | 'left' | 'right' Align the connection point with the first link path end segment opposite point.
alignOffset number Offset the connection point in the direction of the alignment.

Example:

link.source(model, {
    connectionPoint: {
        name: 'anchor',
        args: {
            offset: 10
        }
    }
});
