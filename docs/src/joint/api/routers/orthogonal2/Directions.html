Available values for the sourceDirection and targetDirection options:

orthogonal2.Directions.AUTO Determines the direction depending on the side of the element where the anchor is located.
orthogonal2.Directions.LEFT Sets the direction of the link to the left side.
orthogonal2.Directions.RIGHT Sets the direction of the link to the right side.
orthogonal2.Directions.TOP Sets the direction of the link to the top side.
orthogonal2.Directions.BOTTOM Sets the direction of the link to the bottom side.
