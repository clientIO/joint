The 'orthogonal2' router returns a route with orthogonal line segments just like 'orthogonal' router, except it ignores all vertices and constructs the path in compliance with spacing value. This router does not avoid obstacles. The behavior of the router can be modified by passing optional arguments:

spacing number The minimum distance between the route and the element. Default is 20.
sourceDirection orthogonal2.Directions The direction of the link from the source anchor. Default is orthogonal2.Directions.AUTO
targetDirection orthogonal2.Directions The direction of the link from the target anchor. Default is orthogonal2.Directions.AUTO

Example:

link.router('orthogonal2', {
    spacing: 10,
    sourceDirection: orthogonal2.Directions.TOP,
    targetDirection: orthogonal2.Directions.BOTTOM
});
