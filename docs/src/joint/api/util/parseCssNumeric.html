util.parseCssNumeric(value)

Parse the provided value as a float and return an object with the number as a value parameter. Example:

joint.util.parseCssNumeric(123.4); // => { value: 123.4 }
joint.util.parseCssNumeric('123.4'); // => { value: 123.4 }
util.parseCssNumeric(value, restrictUnits)

If the provided value is a string and restrictUnits is an array of strings, return null unless the unit part of value matches one of the elements of restrictUnits. Then, an object is returned with the number as a value and the unit as a unit parameters. Example:

joint.util.parseCssNumeric('123.4', ['px']); // => null
joint.util.parseCssNumeric('123.4px', ['mm']); // => null
joint.util.parseCssNumeric('123.4px', ['px']); // => { value: 123.4, unit: 'px' }
joint.util.parseCssNumeric('123.4px', ['px', 'mm']); // => { value: 123.4, unit: 'px' }
joint.util.parseCssNumeric('123.4mm', ['px', 'mm']); // => { value: 123.4, unit: 'mm' }
