The devs plugin provides you objects with predefined input and output ports and ready to use example with ports. The plugin consists of basic rectangle objects (Atomic, Coupled) and connection objects (Link).

Devs demo:

joint.shapes.devs.Atomic

The Atomic is inherited rectangle object with in and out ports.

The Atomic attributes:

inPorts array of strings with in ports
outPorts array of strings with out ports

Usage:

var atomic = new joint.shapes.devs.Atomic({

    position: {
        x: 100,
        y: 100
    },
    inPorts: ['in 1', 'in 2'],
    outPorts: ['out 1', 'out 2']
});
The inPorts and the outPorts is possible also set in this way: 
atomic.set('inPorts', ['in 1', 'in 2']);
atomic.set('outPorts', ['out 1', 'out 2']);
The Atomic object is possible add to graph with addCell or addCells functions.

Usage:

graph.addCells([atomic, coupled]);

joint.shapes.devs.Coupled

The Coupled is inherited rectangle object with in and out ports. The difference between Atomic and Coupled is only in default color and size.

Usage:

var coupled = new joint.shapes.devs.Coupled({

    position: {
        x: 100,
        y: 100
    },
    inPorts: ['in'],
    outPorts: ['out']
});

Usage embed that put the atomic to the coupled object.:

coupled.embed(atomic);

Usage fitEmbeds that it fits all the embedded elements inside it.:

coupled.fitEmbeds();

joint.shapes.devs.Link

The devs.Link as the only one inherits from joint.dia.Link.

Usage:

var link = new joint.shapes.devs.Link({
    source: {
        id: sourcePortId,
        port: sourcePort
    },
    target: {
        id: targetPortId,
        port: targetPort
    }
});

Usage addTo that put link to graph.

link.addTo(graph).reparent();
