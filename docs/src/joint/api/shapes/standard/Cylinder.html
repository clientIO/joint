A cylinder with a label.

Supported attrs properties

Selector Node Description
root SVGGElement Container of all nodes
body SVGPathElement Lateral area of the cylinder

The shape has a custom attribute lateralArea set to 10 by default. The attribute sets the SVGPathElement's d in the shape of the lateral area of a tilted cylinder. The attribute expects a tilt value; either a percentage or a number. The value determines the vertical radius of the exposed area of cylinder base; either relative to shape height, or directly. (Refer to the cylinder.tilt function for more detailed specification.)

top SVGEllipseElement Top of the cylinder (cylinder base)
label SVGTextElement Text inside the body
shapes.standard.Cylinder.prototype.tilt
cylinder.tilt()

Return the current cylinder tilt (the value of the body/lateralArea attribute; 10 by default).

cylinder.tilt(tilt, [opt])

Set the cylinder tilt.

If the value provided is a percentage, it determines the vertical radius of the visible area of cylinder base, relative to the height of the refBBox of the shape. If the value is between '0%' and '50%', the top base of the cylinder is exposed. If the value is between '50%' and '100%', the bottom base of the cylinder is exposed. Values lower than '0%' and higher than '100%' are wrapped around to the 0-100 range.

If the value provided is a number, it directly determines the vertical radius of the visible area of cylinder base. If the value is between 0 and half of refBBox height, the top base of the cylinder is exposed. If the value is between half of refBBox height and its full height, the bottom base of the cylinder is exposed. Values outside the range are wrapped around to the accepted range.

Cycling through values '0%' to '100%' (or corresponding non-relative values) produces an appearance of a cylinder rotating by 180 degrees around the y-axis. All special cases are covered; at '0%' and '100%', a side view of the cylinder is presented - a rectangle. At '50%', a top-down view of the cylinder is presented - an ellipse.

The function automatically sets the value of several attributes: body/lateralArea; top/ry, top/cy, top/refRy and top/refCy; and label/refY and label/refY2. If custom values are required for these arguments, make sure to assign them only after cylinder tilt is set.

Example usage:

var cylinder = new standard.Cylinder();
cylinder.resize(100, 200);
cylinder.position(525, 75);
cylinder.attr('root/title', 'joint.shapes.standard.Cylinder');
cylinder.attr('body/fill', 'lightgray');
cylinder.attr('top/fill', 'gray');
cylinder.attr('label/text', 'Cylinder');
cylinder.tilt('10%');
// custom label position:
cylinder.attr('label/refY', '100%');
cylinder.attr('label/refY2', 10);
cylinder.attr('label/textVerticalAnchor', 'top');
cylinder.addTo(graph);
