An ellipse with an image and label.

Supported attrs properties

Selector Node Description
root SVGGElement Container of all nodes
image SVGImageElement Image of the shape
border SVGEllipseElement Border around the icon
background SVGEllipseElement Area behind the icon
label SVGTextElement Text below the icon 
var icon = new joint.shapes.standard.OvalImage();
icon.resize(150, 100);
icon.position(225, 410);
icon.attr('root/title', 'joint.shapes.standard.OvalImage');
icon.attr('label/text', 'Oval Image');
icon.attr('border/strokeWidth', 5);
icon.attr('background/fill', 'lightgray');
icon.attr('image/xlinkHref', 'image.png');
icon.addTo(graph);
