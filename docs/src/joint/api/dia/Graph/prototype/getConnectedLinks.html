graph.getConnectedLinks(element, opt)

Get all the associated links to the element.

If opt.inbound === true, only inbound links will be returned. Similarly, if opt.outbound === true, only outbound links will be returned.

If opt.deep === true, all outside links that connect to any of the element's descendants (embedded and deeply embedded elements) are returned, as well. Additionally, if opt.includeEnclosed === true, also all the links completely enclosed within element (i.e. links connecting two embedded descendants) will be returned.
