linkView.addLabel(label)

Add a new label to the link.

Normally, labels are set through the link.addLabel function. However, access to linkView functions is sometimes needed while adding new labels to the labels array.

This function is useful within custom linkView event listener definitions. For example, if one wanted to add a new label on the default click event, instead of a vertex:

var paper = new joint.dia.Paper({
    // ...
    linkView: joint.dia.LinkView.extend({
        // overwrite default event listener with custom functionality
        pointerdown: function(evt, x, y) {
            if (V(evt.target).hasClass('connection') || V(evt.target).hasClass('connection-wrap')) {
                this.addLabel({
                    position: this.getClosestPointNormalizedLength(new g.Point(x, y)),
                    attrs: { text: { text: '*' }}
                });
            }
        }
    })
});
