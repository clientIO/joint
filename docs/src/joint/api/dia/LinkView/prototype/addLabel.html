linkView.addLabel(x, y [, opt])

Add a new default label to the link at the coordinates provided. If you need to add a label with custom markup or properties, use the link.addLabel() function instead.

This method uses the link.getLabelPosition() function to determine the new label's position. By default, position.distance is recorded relative to connection length (as a number in the [0,1] range), and position.offset is set relative to the connection (as a number). This behavior may be changed by providing an opt object with some of the accepted boolean flags:

  • absolute: true records distance absolutely (as distance from beginning of link)
  • reverse: true switches distance to be calculated from end of link, if absolute
  • absoluteOffset: true records offset absolutely (as x and y from connection)

This function is useful within custom linkView event listener definitions. The following example adds a new label on a pointerdown event, instead of a vertex (which is the default behavior):

var paper = new joint.dia.Paper({
    // ...
    linkView: joint.dia.LinkView.extend({
        // overwrite default event listener with custom functionality
        pointerdown: function(evt, x, y) {
            this.addLabel(x, y);
        }
    })
});
