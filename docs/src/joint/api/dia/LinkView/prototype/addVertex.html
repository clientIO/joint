linkView.addVertex(vertex)

Add a new vertex (an object with x and y properties) to the link.

Normally, vertices are set through the link.addVertex function. However, sometimes it is useful to let the linkView automatically determine the index of the new vertex in the vertices array.

The linkView checks the distance (from the beginning of the path, along the path) of vertices in the vertices array and compares it the to the distance of the new vertex. The new vertex is inserted ahead of the first farther vertex in the vertices array.

This function is useful within custom linkView event listener definitions. For example, if one wanted to add a new vertex on double click instead of the default click event:

var paper = new joint.dia.Paper({
    // ...
    interactive: function(cellView) {
        if (cellView.model instanceof joint.dia.Link) {
            return { vertexAdd: false }; // disable default vertexAdd pointerdown functionality
        }
        return true;
    },
    linkView: joint.dia.LinkView.extend({
        // add custom pointerdblclick functionality
        pointerdblclick: function(evt, x, y) {
            if (V(evt.target).hasClass('connection') || V(evt.target).hasClass('connection-wrap')) {
                this.addVertex({ x: x, y: y });
            }
        }
    })
});
