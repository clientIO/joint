link.target()

Return a shallow copy of the target property of the link.

If the end of the link is connected to an element, an object in the format { id: elementID } is returned. If the end of the link is not connected to an element (it is "pinned" to the paper at a point), an object in the format { x: targetX, y: targetY } is returned instead.

If you need to be sure that an Element is returned (and not a Point), use the link.getTargetElement function instead.

link.target(target [, opt])

Set the target of the link.

If the link is to be connected to an element, send an object in the format { id: element.id }. If the link is to be pinned to the paper, send an object in the format { x: targetX, y: targetY } (or a g.Point object).
