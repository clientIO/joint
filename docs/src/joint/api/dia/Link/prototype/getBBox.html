link.getBBox()

Return the bounding box of the link model represented as a g.Rect.

This method creates the bounding box of the link from its vertices. This simplified heuristic may cause portions of curved links to protrude out of the reported bounding box. If you definitely need to make sure to include all of the link view, consider using the linkView.getBBox function instead. See the documentation of the elementView.getBBox function for an explanation of the limitations of this approach (albeit in the context of element views).
