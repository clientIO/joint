paper.dumpViews([opt])

Add all paper views into the DOM.

Several extra arguments may be provided in the opt object:

batchSize number For async papers, how many views should there be per one asynchronous process? Default is 1000.
viewport function Callback function to determine whether a given view should be added to the DOM. By default, paper.options.viewport is used. The callback function is provided with three arguments:
view mvc.View The view in question
isDetached boolean Is the view currently hidden from the paper?
paper dia.Paper This paper (for context)
progress function Callback function that is called whenever a view is added to the paper. The callback function is provided with three arguments:
done boolean Are we done?
processed number How far along are we (absolute numbers)?
total number How many views are there (absolute numbers)?
