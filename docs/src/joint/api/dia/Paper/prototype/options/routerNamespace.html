routerNamespace - this option allows the user specify a custom link router namespace right in the paper options. This works in tandem with thedefaultRouter option. If a user provides an object { name: 'customRouter' } as the defaultRouter, the paper will look for this router in the provided routerNamespace. The default router used is 'normal'. 

    const customNamespace = {};

    new joint.dia.Paper({
        defaultRouter: { name: 'customRouter' },
        routerNamespace: customNamespace,
    });

    // the paper looks for the router under customNamespace['customRouter']
}
