interactive - if set to false, interaction with elements and links is disabled. If it is a function, it will be called with the cell view in action and the name of the method it is evaluated in ('pointerdown', 'pointermove', ...). If the returned value of such a function is false interaction will be disabled for the action. For links, there are special properties of the interaction object that are useful to disable the default behaviour.

In the table bellow there is a list of properties followed by an example with the related property set to false.

arrowheadMove vertexMove
vertexAdd vertexRemove
labelMove useLinkTools

For a comparison this example has default settings - all interactions on the link are enabled.
