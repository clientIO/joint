drawGrid - option whether the paper grid should be drawn or not. It could be a boolean or an object. It defaults to false
color string color of the default grid
thickness number thickness of the default grid
gridPattern string you can define your own custom grid pattern. gridPattern is a single tile which the resulting grid consists of. It can be a valid SVG element. For example <rect width="100%" height="100%" stroke="red" stroke-width="1" fill="transparent"/>
updateGridPattern function(vel, opt) this callback function is called on every re-draw of the grid. There can be many reasons to re-draw the grid. For example scaling the paper changes the physical grid size. vel and opt is passed into this callback function, where the vel is the Vectorizer element, opt contains information about actual grid size - width, height and transformation: scale and translate - sx, sy, ox, oy. You are able update attributes on your gridPattern shape to adapt it to actual transformations.
