paper.requireView(cell)

Ensure that the view associated with the cell model is part of the paper.

This function finds the view by the cell model. If the view is not part of the paper (e.g. because it was outside the paper viewport), the function makes it visible. This is necessary when calling certain CellView methods (i.e. elementView.getBBox or linkView.getBBox) - these functions need the view to be present in the DOM to function properly.
