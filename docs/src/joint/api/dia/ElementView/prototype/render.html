elementView.render()

The method renders the markup, updates the presentation attributes and resizes, rotates and translates the view. This method is invoked when the element view is added to the paper and also when the attrs attribute is changed together with the dirty flag being passed along. 

// Render is called here when the graph is connected to a paper.
model.addTo(graph);
// Causing the element view to re-render itself.
model.attr('rect/fill', 'red', { dirty: true })
