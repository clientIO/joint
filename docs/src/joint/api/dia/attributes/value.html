The value is not rendered as part of the graphics, but it's used to set the value of the form control element (used typically with an input inside the foreignObject).

element.attr('input/value', 'Text');

When setting a value of the select element, pass the value of the option element.

element.attr('select/value', 'option1');

For the select with multiple options, pass an array of values.

element.attr('select/value', ['option1', 'option2]', { rewrite: true });
